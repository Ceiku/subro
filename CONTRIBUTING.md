```text

```

## How to contribute

1. Open an issue or discuss the change
2. Fork and branch from `main`
3. Run `./bin/doctor` after broker/skill changes
4. Open a pull request with a clear description

## Security

Do not open public issues for sensitive vulnerabilities. Contact the repository
owner privately via GitHub.

## Code of conduct

Be respectful and constructive. We reserve the right to reject contributions
that do not align with project goals or license terms.
